Image Crawler Completable Future App (Assignment 4)

The purpose of this assignment is to deepen your experience using Java 8 functional programming features and their integration with Java 8 completable futures. In particular, in this assignment you'll develop an app that uses the Java 8 completable futures framework to obtain, store, and transform images.

The code you'll write for this assignment will have GUI and will run as an app. Feel free to use the GUI to help debug your code.

Getting Started

Overview of the Assignment 4

In this part of the assignment you'll complete the implementation of an image crawler class in the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/crawlers/CompletableFutureCrawler1.java file that obtains, stores, and transforms images. This class uses Java 8 completable futures to perform an "image crawl" starting from a root Uri. Depending on the parameters used to run the tests, images from HTML page reachable from the root Uri are either downloaded from a remote web server or read from the local file system and the results are stored in files. You can also enable/disble transforms via the GUI.

The bulk of the code for the app is provided in the skeletons. In fact, a complete solution using Java 7 features provided in the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/crawlers/SequentialLoopsCrawler.java file. Your will therefore focus on converting this Java 7-based class to the CompletableFutureCrawler1 class that uses Java 8 completable futures.

Everyone taking the course (i.e., both ugrads and grads) must also complete implementations of all the methods in the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/utils/ArrayCollector.java, image-crawler/src/main/java/edu/vanderbilt/imagecrawler/utils/UnsynchronizedArray.java, and image-crawler/src/main/java/edu/vanderbilt/imagecrawler/utils/SynchronizedArray.java files.

Your solution should use no loops in this assignment and instead use Java 8 completable future methods.
