Image Crawler Completable Future App (Assignment 4)

The purpose of this assignment is to deepen your experience using Java 8 functional programming features and their integration with Java 8 completable futures. In particular, in this assignment you'll develop an app that uses the Java 8 completable futures framework to obtain and store images.

Like the earlier assignments, the code you'll write here will have GUI and will run as an Android app. Please use this GUI along with the JUnit tests discussed below to help test and debug your solution.

Getting Started

Overview of the Assignment 4

In this part of the assignment you'll complete the implementation of an image crawler class in the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/crawlers/CompletableFutureCrawler.java file that obtains and stores images. This class uses Java 8 completable futures to perform an "image crawl" starting from a root Uri. Depending on the parameters used to run the tests, images from HTML page reachable from the root Uri are either downloaded from a remote web server or read from the local file system and the results are stored in files.

The bulk of the code for the app is provided in the skeletons. In fact, a complete solution using Java 7 features provided in the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/crawlers/SequentialLoopsCrawler.java file. Your will therefore focus on converting this Java 7-based class to the CompletableFutureCrawler class that uses Java 8 completable futures. This SequentialLoopsCrawler implementation differs somewhat from the version provided in previous assignments, so you'll need to revise portions of your Java 8 solution from earlier assignments to account for these changes.

Everyone taking the course (i.e., both ugrads and grads) must also include your completed implementations of all the methods in the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/utils/ArrayCollector.java and image-crawler/src/main/java/edu/vanderbilt/imagecrawler/utils/Array.java files.

Your solution should use no loops in this assignment and instead use Java 8 completable future methods.

Skeleton Code

Skeleton code for this assignment is available here in my GitHub account. Now that you've setup your GitLab account you can pull this skeleton code in your repository, read it carefully, and complete the "TODO" markers.

Unit Tests

JUnit unit tests in the image-crawler/src/test/java/edu/vanderbilt/imagecrawler folder are provided to increase our confidence that your implementation is working as expected. You'll need to learn how to use the Android Studio GUI to run the JUnit test locally on your computer, as described here. As usual, testing only demonstrates the presence of bugs, not their absence, so don't rely solely on the JUnit tests to detect problems in your code.

Concluding Remarks

Assignment 4 is more complicated than the previous assignments. In particular, the asynchronous nature of completable futures may take some time to get accustomed to, so please start the assignment soon and ask questions in class, office hours, and on the discussion forums.
