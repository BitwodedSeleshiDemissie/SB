Array (Assignment 1 Final Submission)

The purpose of this assignment is to deeper your experience using Android Studio to develop and test a Java Array class written using Java 8 functional programming features, such as functional interfaces. Although we'll be using Android Studio, the code you'll write for this assignment will not have GUI and will not run as an app. It's important that your Array implementation work properly since it's required by all subsequent assignments in this course, including the ones that are Android apps.

Getting Started

Before you complete Assignment 1 you'll need to do the following things:

  • Integrate your solution into the new skeletons, which will require some minor changes to the ArrayIterator.next() method to throw NoSuchElementException. Also, note that there's no more ugrad vs. grad folders, i.e., everything is in a single directory structure.

  • Update your assignment 1a and 1b solutions to address my comments, as well as ensuring that you've address any other relevant comments from the ``frequently made mistakes'' videos I've posted to my youtube playlist.

  • Ensure that your integrated solution passes the latest/greatest automated unit tests (see below) that are part of the assignment 1 final submission project.

Skeleton Code

Skeleton code for this assignment is available here in my GitHub account. Once you've setup your GitLab account you can pull this skeleton code in your repository, read it carefully, and complete the "TODO" markers. Please note that you'll need to complete the "TODO" markers in both the Array.java and Assignment.java files this time.

Unit Tests

JUnit unit tests in the app/src/test/java/edu/vandy/imagecrawler folder are provided to increase our confidence that your implementation is working as expected. You'll need to learn how to use the Android Studio GUI to run the JUnit test locally on your computer, as described here. As usual, testing only demonstrates the presence of bugs, not their absence, so don't rely solely on the JUnit tests to detect problems in your code.

Concluding Remarks

This assignment is intentionally designed to be easy to help you get familiar with installing/using Java and Android Studio, as well as some basic Java 8 functional programming features. If you find it's extremely hard to complete this assignment you may not be ready to take this class. If this is the case, please meet with me ASAP to discuss your options.
