Overview of the Array Assignment

The purpose of this assignment is to give you experience using Android Studio to develop and test a Java Array class. An Array is an abstract data type (ADT) implemented via a single contiguous buffer that provides operations like get(), set(), add(), remove(), and size(). Your task is to implement a generic Array class in Java. This class will be different from Java built-in arrays in various ways, e.g., it can be resized and used in conjunction with other Java Collection Framework classes.

Graduate students (or students who are taking this course for graduate credit) must do the following additional work:

  • Implement the requirements of the Iterator interface.

As much as possible, you should avoid using loops in this assignment and instead try to use Arrays.copyOf() (this isn't always possible, of course).

The following resources may be helpful in completing this assignment:

  • java.utils.Arrays documentation.
  • The Oracle Java Tutorial

If you're Java knowledge is rusty you might consider watching the videos in our Coursera Java for Android MOOC.
