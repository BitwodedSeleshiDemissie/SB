Image Crawler Stream App (Assignment 2a)

The purpose of the first part of this two-part assignment is to deepen your experience using Java 8 functional programming features and their integration with Java 8 sequential streams. In particular, in this assignment you'll develop an app that uses the Java 8 streams framework to obtain and store images.

Although we'll be using Android Studio, the code you'll write for this assignment will not have GUI and will not run as an app.

Getting Started

Before you complete the initial submission of Assignment 2a you'll need to do the following:

  • Complete your implementation of assignment 1b, taking into account my review comments. In particular, it's important that your Array implementation works properly since it's used as the basis for the app in this assignment. Please come to office hours if you have any questions about assignment 1.

Overview of the Assignment 2a

In this part of the assignment you'll complete the implementation of an image crawler class in the app/src/main/java/edu/vanderbilt/crawlers/SequentialStreamsCrawler.java file that obtains and stores images. This class uses Java 8 sequential streams to perform an "image crawl" starting from a root Uri. Depending on the parameters used to run the tests, images from HTML page reachable from the root Uri are either downloaded from a remote web server or read from the local file system and the results are stored in files.

The bulk of the code for the app is provided in the skeletons. In fact, a complete solution using Java 7 features provided in the app/src/main/java/edu/vanderbilt/crawlers/SequentialLoopsCrawler.java file. Your will therefore focus on converting this Java 7-based class to the SequentialStreamsCrawler class that uses Java 8 sequential streams.

Undergraduate students must do the following additional work:

  • Implement the key methods in the Spliterator interface in the app/src/main/java/edu/vanderbilt/utils/Array.java class (which was done by graduate students in assignment 1b). You'll need these methods for assignment 2b.

Likewise, graduate students (or students who are taking this course for graduate credit) must do the following additional work:

  • Complete the implementation of the class in the app/src/main/java/edu/vanderbilt/utils/ArrayCollector.java file and use it in the implementation of the crawlHyperLinksOnPage() method. Undergraduates can optionally implement the ArrayCollector class or use a different implementation (e.g., a forEach() loop).

Your solution should use no loops in this assignment and instead use Java 8 sequential stream aggregate operations.

Unit Tests

JUnit unit tests in the app/src/test/java/edu/vanderbilt/imagecrawler folder are provided to increase our confidence that your implementation is working as expected. You can run all the unit tests by right-clicking on 'AllTests' in this folder. If you need help learning how to use the Android Studio GUI to run the JUnit test locally on your computer please see here and/or ask questions. As usual, testing only demonstrates the presence of bugs, not their absence, so don't rely solely on the JUnit tests to detect problems in your code.
